## 🚀 Getting Started

### Installation

1. **Download** the theme ZIP file from this repository:
   ```
   glovenix_shopify_theme.zip
   ```

2. **Log in** to your Shopify Admin Panel.

3. Navigate to **Online Store → Themes**.

4. Click **"Add theme"** → **"Upload zip file"**.

5. Select the downloaded `glovenix_shopify_theme.zip` and click **Upload**.

6. Once uploaded, click **"Customize"** to start editing your store.

## 🛠️ Customization

After installation, you can customize the theme directly from the Shopify Theme Editor:

| Section | What You Can Edit |
|---|---|
| **Header** | Logo, navigation, announcement bar |
| **Hero Banner** | Images, videos, headlines, CTA buttons |
| **Product Pages** | Layout, media, tabs, reviews |
| **Collections** | Grid/list view, filters, sorting |
| **Footer** | Links, newsletter, social icons |
| **Colors & Fonts** | Brand colors, typography, button styles |

## 📁 File Structure

```
glovenix_shopify_theme/
├── assets/          # CSS, JS, images, fonts
├── config/          # Theme settings schema
├── layout/          # Base theme layout (theme.liquid)
├── locales/         # Translation files
├── sections/        # Customizable page sections
├── snippets/        # Reusable Liquid code snippets
└── templates/       # Page templates (product, collection, etc.)
```

## 📋 Requirements

- Shopify Basic plan or higher

## 💻 Local Development (Optional)

For developers who want to work locally:

```bash
# Install Shopify CLI
npm install -g @shopify/cli @shopify/theme

# Connect to your store
shopify theme dev --store=your-store.myshopify.com
```
